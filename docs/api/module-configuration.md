# Module Configuration

Nuxt Kirby can be configured via the `kirby` property in your `nuxt.config.ts` file.

```ts
// `nuxt.config.ts`
export default defineNuxtConfig({
  modules: ['nuxt-kirby'],

  kirby: {
    // ... Your options here
  }
})
```
