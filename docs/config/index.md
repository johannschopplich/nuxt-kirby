# Module Configuration

## Usage

Adapt Nuxt KQL to your needs by setting module options in your `nuxt.config.ts`:

```ts
// `nuxt.config.ts`
export default defineNuxtConfig({
  modules: ['nuxt-kql'],

  kql: {
    // ... Your options here
  }
})
```
